Instructions

This page helps test the admin authentication flow:

  1. Open this file directly in your browser
  2. Click "Login to Admin" to set the auth cookie
  3. Click "Check Auth Status" to verify you're authenticated
  4. Click "Go to Admin Dashboard" to access the admin area
